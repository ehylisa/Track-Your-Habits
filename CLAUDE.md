# HabitTracker

An iOS app for tracking daily habits and streaks, built with SwiftUI.

## Goal

A simple, clean habit tracker where users can add habits, mark them complete each day, and see their current streak.

## Current State

- `ContentView.swift` — main screen with title, empty state message, and "Add Habit" button (no functionality yet)
- No data model or persistence yet
- No navigation or additional screens yet

## Planned Features

- Add/delete habits
- Mark habits complete for the day
- Streak tracking per habit
- Persist habits across launches (SwiftData or UserDefaults)
